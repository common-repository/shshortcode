Will generate Shortcodes for use with SyntaxHighlighter extended and similar plugins

== Description ==

This plugin must be used along side SyntaxHighlighter extended or plugins that offer the same functionality (Although this plugin cannot be garenteed to work with other plugins). 

Once installed in allows users to generate shortcodes easily by setting multiple options.

== Installation ==

1. Extract the files from SHshortcode.zip
1. Upload the files from the extracted directory to '/wp-content/plugins' directory
1. Log into your Wordpress control panel and activate the SHshortcode plugin
1. When posting a new page or post, click on the button labelled 'SH' which will bring up a window allowing you to select options and generate a Shortcode
